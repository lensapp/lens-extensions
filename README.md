## Vendor Extensions

- [Mirantis Container Cloud](https://github.com/Mirantis/lens-extension-cc/) by [Mirantis](https://mirantis.com)

- [GKE Sync](https://github.com/jakolehm/lens-extension-gke-sync) - Sync GKE projects & clusters automatically
